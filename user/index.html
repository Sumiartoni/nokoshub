🆘 Bantuan

Panduan singkat pembelian OTP, refund, dan hal penting yang perlu diperhatikan saat order.

📱
Cara membeli nomor OTP
1
Pilih server dan layanan

Tentukan jalur server yang ingin dipakai, lalu pilih layanan OTP seperti WhatsApp, Telegram, Shopee, dan lainnya.

2
Pilih negara lalu buat nomor

Pilih negara yang tersedia, sistem akan menampilkan harga aktif lalu membuat nomor virtual secara otomatis.

3
Tunggu OTP masuk

Setelah nomor berhasil dibuat, tunggu kode OTP masuk ke dashboard. Tidak perlu refresh halaman secara terus-menerus.

💸 Kapan saldo akan direfund?
Jika OTP tidak masuk sampai batas waktu tunggu berakhir, saldo order akan direfund otomatis ke akun Anda.
  • Refund berlaku untuk pesanan yang gagal menerima OTP sesuai flow sistem.
  • Jika provider menandai order gagal atau dibatalkan, saldo akan kembali otomatis.
  • Refund biasanya akan langsung terlihat di saldo dan riwayat transaksi.
⏳ Kenapa OTP belum masuk?
  • Layanan tujuan sedang lambat mengirim SMS OTP.
  • Nomor dari server yang dipilih sedang padat atau delay.
  • App tujuan meminta verifikasi tambahan selain OTP SMS.
  • Provider upstream sedang gangguan sementara.
Tips: tunggu sampai timer selesai dan jangan buru-buru menutup halaman saat order masih aktif.
📌 Hal penting saat order OTP
  • Pastikan saldo akun cukup sebelum membuat order.
  • Jangan gunakan nomor yang sama untuk beberapa form sekaligus.
  • Jika server 1 penuh atau lambat, Anda bisa coba server 2.
  • Cek menu Pesanan Saya untuk memantau status dan OTP yang masuk.
❓ Tanya Jawab Singkat
Apakah OTP selalu masuk cepat?

Tidak selalu. Kecepatan OTP bergantung pada layanan tujuan dan kondisi server/provider saat itu.

Apakah saldo hangus kalau OTP tidak masuk?

Tidak. Jika order gagal atau OTP tidak diterima sesuai ketentuan sistem, saldo akan direfund otomatis.

Di mana saya melihat refund?

Refund akan terlihat di saldo akun dan juga tercatat di menu Transaksi.
