❓ FAQ

Pertanyaan yang Sering Ditanyakan

Tidak menemukan jawaban yang Anda cari? Hubungi tim support kami!

Apa itu nokos WA atau nomor kosong WhatsApp?
+

Nokos WA adalah nomor virtual untuk menerima SMS OTP WhatsApp tanpa memakai nomor pribadi. Di NokosHUB, Anda bisa memilih layanan WhatsApp, menggunakan nomor kosong yang tersedia, lalu melihat kode OTP langsung dari dashboard.

Apa bedanya nomor virtual, nomor kosong, dan OTP virtual?
+

Nomor virtual atau nomor kosong adalah nomor sementara untuk menerima kode verifikasi. OTP virtual adalah kode SMS OTP yang masuk ke nomor tersebut dan ditampilkan otomatis agar proses verifikasi akun lebih cepat.

Apa itu OTP virtual dan bagaimana cara kerjanya?
+

OTP (One Time Password) virtual adalah kode verifikasi yang dikirim ke nomor telepon virtual yang kami sediakan. Anda menggunakan nomor ini untuk mendaftarkan akun di berbagai aplikasi, kemudian OTP yang diterima akan langsung tampil di dashboard NokosHUB Anda dalam hitungan detik.

Berapa lama OTP berlaku setelah diterima?
+

OTP yang diterima biasanya berlaku selama 5-10 menit tergantung platform yang Anda gunakan. Kami menyarankan untuk segera menggunakan OTP setelah diterima. Jika OTP expired, Anda bisa request OTP baru tanpa biaya tambahan (saldo tetap utuh).

Apakah saldo yang saya isi bisa kadaluarsa?
+

Tidak! Saldo NokosHUB Anda tidak memiliki masa kadaluarsa. Anda bisa menggunakannya kapan saja sesuai kebutuhan. Kami berkomitmen untuk menjaga saldo Anda tetap aman selama akun aktif.

Apa yang terjadi jika OTP tidak diterima?
+

Jika OTP tidak diterima dalam 5 menit, saldo Anda akan otomatis dikembalikan 100%! Kami memiliki sistem refund otomatis yang memastikan Anda tidak rugi apabila terjadi kegagalan pengiriman OTP.

Apakah penggunaan NokosHUB aman dan legal?
+

NokosHUB beroperasi sesuai dengan ketentuan yang berlaku. Platform kami dirancang untuk kemudahan verifikasi akun pribadi yang sah. Kami tidak mendukung penggunaan layanan kami untuk aktivitas ilegal atau penipuan. Data privasi Anda terlindungi dengan sistem enkripsi SSL 256-bit.

Apakah tersedia API untuk integrasi bisnis?
+

Ya! Kami menyediakan REST API yang lengkap dengan dokumentasi detail untuk integrasi ke sistem bisnis Anda. API tersedia untuk semua pengguna akun Business. Hubungi tim kami untuk informasi lebih lanjut tentang akses API dan harga khusus untuk volume besar.
